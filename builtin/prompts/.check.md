You are checking {{ language }} code against the following rule:

---
{{ rule_content }}
---


File: {{ target_path }}

Here is the context of the file being checked:

```{{ language }}
{{ target_content }}
```

Analyze this file against the rule above.

## Response Format

If the rule is not applicable to this file type OR if the rule is followed, respond with exactly:

PASS

If violations are found, respond with the following format for EACH violation:

VIOLATION
Rule: {{ rule_name }}
File: {{ target_path }}
Line: [line number]
Severity: [error/warning/info/hint]
Message: [description of the violation]
Suggestion: [suggested fix]

Be specific about line numbers where violations occur, provide actionable suggestions for fixing violations, and use the severity level appropriate to the violation's impact.
