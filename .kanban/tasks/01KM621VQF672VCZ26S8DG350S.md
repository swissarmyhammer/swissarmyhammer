## What

Two related changes to `EntityContext` in `swissarmyhammer-entity/src/context.rs`:

### 1. Move `.trash/` under each entity type directory

**Before:** `{root}/.trash/{type}s/` (e.g. `.kanban/.trash/tasks/`)
**After:** `{root}/{type}s/.trash/` (e.g. `.kanban/tasks/.trash/`)

Each entity type's directory becomes self-contained — live, trashed, and archived files all live under the same parent.

Changes:
- `trash_dir()` — returns `{root}/{type}s/.trash/` instead of `{root}/.trash/{type}s/`
- `read_changelog_with_trash_fallback()` — update trash path construction
- All existing tests that assert on trash paths

**Migration:** Add a one-time migration in `EntityContext` (or a standalone function) that moves files from the old `{root}/.trash/{type}s/` to `{root}/{type}s/.trash/` if the old layout exists. Run on construction or first access.

### 2. Add `.archive/` support (new)

Same layout pattern as the relocated `.trash/`:
- `archive_dir(&self, entity_type) -> PathBuf` — returns `{root}/{type}s/.archive/`
- `archive(&self, entity_type, id) -> Result<Option<ChangeEntryId>>` — reads entity, appends an "archive" changelog entry, moves data file + changelog to `{type}s/.archive/`. Entity no longer returned by `list()`.
- `unarchive(&self, entity_type, id) -> Result<Option<ChangeEntryId>>` — moves files from `.archive/` back to live dir, appends "unarchive" changelog entry.
- `list_archived(&self, entity_type) -> Result<Vec<Entity>>` — reads from `{type}s/.archive/` using same `io::read_entity_dir()` + compute pipeline.
- `read_archived(&self, entity_type, id) -> Result<Entity>` — reads a single archived entity.

### Implementation approach

- Reuse `io::trash_entity_files()` and `io::restore_entity_files()` — they take source/dest paths, directory name doesn't matter
- `archive()` mirrors `delete()` but targets `.archive/` and uses op "archive"
- `unarchive()` mirrors `restore_from_trash()` pattern

### Files
- `swissarmyhammer-entity/src/context.rs` — relocate trash_dir, add archive methods, add migration
- `swissarmyhammer-entity/src/io.rs` — if any path construction is hardcoded there
