Now that the backend handles computed field writes via `entity.update_field`, remove all the frontend body-manipulation hacks from `multi-select-editor.tsx`.\n\n## Current state\nThe multi-select editor has two code paths:\n1. **Reference fields** (assignees): commits array of IDs via `onCommit(selectedIds)` — works correctly\n2. **Computed tags** (`isComputedTags`): reads task body, does string manipulation (`#tag` append/remove), calls `entity.update_field` on the body — THIS IS THE HACK\n\n## New state\nBoth paths commit the same way: `onCommit(selectedIds)`. The grid's `handleCommit` calls `updateField(entity_type, id, field_name, value)` which sends `entity.update_field` to the backend. The backend sees the field is computed, routes through `DeriveHandler`, and handles body mutation server-side.\n\n## Changes\n- Remove `isComputedTags` branching in `addItem`, `removeItem`, and `commit`\n- Remove direct `invoke(\"dispatch_command\", ...)` calls for body mutation\n- Remove `getStr(entity, \"body\")` usage\n- `addItem` just adds to `selectedIds` (same as reference path)\n- `removeItem` just removes from `selectedIds` (same as reference path)\n- `commit` always calls `onCommit(selectedIds)` (never `onCancel`)\n- Tags field now uses `editor: multi-select` from schema, so frontend auto-opens the multi-select editor\n\n## Files\n- Edit: `ui/src/components/fields/editors/multi-select-editor.tsx` — simplify to single code path\n- Edit: `ui/src/components/fields/editors/multi-select-editor.test.tsx` — update tag tests (no longer mocking invoke for body updates)\n- May need: `ui/src/lib/schema-context.tsx` — ensure computed fields with `editor: multi-select` are treated as editable\n\n## Checklist\n- [ ] Remove `isComputedTags` branches from addItem, removeItem, commit\n- [ ] Both reference and computed fields use identical commit path\n- [ ] Remove direct invoke calls for body field manipulation\n- [ ] Update tests: tag field commits array via onCommit, not via invoke\n- [ ] Verify assignee editing still works (no regression)\n- [ ] `npx vitest run` passes\n- [ ] `npx tsc --noEmit` passes
