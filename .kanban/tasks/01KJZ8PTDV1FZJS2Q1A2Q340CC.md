Create a hook that collects `contextMenu: true` commands from the scope chain and shows a native popup via Rust.\n\nSince only one context menu can be open at a time, use a module-level pending handlers map.\n\n```ts\n// lib/context-menu.ts\nconst pendingHandlers = new Map<string, CommandDef>();\n\nexport function useContextMenu(): (e: React.MouseEvent) => void {\n  const commands = useAvailableCommands();\n  const contextCommands = commands.filter(c => c.command.contextMenu);\n  \n  return (e: React.MouseEvent) => {\n    e.preventDefault();\n    e.stopPropagation();\n
