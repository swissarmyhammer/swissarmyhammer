## What

Migrate `agent-client-protocol-extras` to ACP 0.11 by porting the simplest wrapper (`TracingAgent`) first — this establishes the new builder/handler pattern that the remaining wrappers reuse.

Files:
- `agent-client-protocol-extras/src/lib.rs` — schema-type imports, exports.
- `agent-client-protocol-extras/src/hook_config.rs` — hook config types (no Agent trait dependency, but uses schema types).
- `agent-client-protocol-extras/src/tracing_agent.rs` — `TracingAgent<A>` wrapper.

The `Agent` trait is gone in 0.11 — replaced by an `Agent` Role marker + `Agent.builder().on_receive_request(...).connect_to(...)` API. Reshape `TracingAgent` accordingly: it should still wrap an inner agent and emit tracing spans/events for every request and notification, but the wrapping mechanism is now the new builder/handler pattern, not `impl Agent for TracingAgent`.
