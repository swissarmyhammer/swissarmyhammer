## What

Create `kanban-cli/src/doctor.rs` — diagnostic checks for kanban setup.

Model on `shelltool-cli/src/doctor.rs`. Implement `KanbanDoctor` using `swissarmyhammer_doctor::{Check, CheckStatus, DoctorRunner}`.

Three checks:
1. **Git Repository** — `find_git_repository_root()`, warning if not in a git repo
2. **kanban in PATH** — check `which kanban` / scan PATH for the binary
3. **Board initialized** — check for `.kanban/board.yaml` in cwd (or git root); warning if missing, with fix: `kanban board init --name "My Board"`
