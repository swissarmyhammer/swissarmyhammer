board-view.tsx defines `handleDragEnd` (line 265) with `persistMove` in its useCallback dependency array (line 353), but `persistMove` is a `const` defined later (line 382). Since `const` is not hoisted like function declarations, accessing it before initialization triggers a ReferenceError (temporal dead zone). Move `persistMove` above `handleDragEnd` to fix. This is a regression from converting the plain `async function persistMove()` (which was hoisted) to `const persistMove = useCallback(...)` (which is not).
