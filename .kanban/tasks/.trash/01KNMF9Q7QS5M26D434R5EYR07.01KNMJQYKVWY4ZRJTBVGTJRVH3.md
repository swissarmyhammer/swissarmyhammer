## What

The `perspective.next` and `perspective.prev` commands work on the backend but produce no visible change in the UI for two reasons:

1. **Keybindings are not registered in the frontend.** The YAML declares `Mod+]`/`Mod+[` (CUA) and `gt`/`gT` (vim) but these have no counterpart in `BINDING_TABLES`/`SEQUENCE_TABLES` in `keybindings.ts`. Pressing these keys does nothing.

2. **`view_kind` arg is not injected.** The backend requires `view_kind` as a required arg, but the keybinding system dispatches commands with no args. Even if the keys were wired, the command would fail with a missing-arg error.

### Files to modify

1. **`kanban-app/ui/src/lib/keybindings.ts`** â€” Add `perspective.next` and `perspective.prev` to `BINDING_TABLES` (for CUA: `Mod+]`/`Mod+[`) and `SEQUENCE_TABLES` (for vim: `gt`/`gT`).

2. **`kanban-app/ui/src/components/perspective-tab-bar.tsx`** or **`kanban-app/ui/src/components/perspective-container.tsx`** â€” Register a frontend-side command handler (via `CommandScopeProvider` or a `useEffect` listener) that intercepts `perspective.next`/`perspective.prev`, reads the current view kind from `useViews()` or the active view context, and dispatches the command to the backend with `{ args: { view_kind: activeView.kind } }`.

   Alternative: modify the backend `cycle_perspective` in `swissarmyhammer-kanban/src/commands/perspective_commands.rs` to derive `view_kind` from `UIState.active_view_id()` instead of requiring it as an arg, eliminating the frontend injection need. This is cleaner if the active view kind can be reliably parsed from the view ID (e.g. strip the trailing ULID suffix).

- The `ui.perspective.set` path (tab clicks) already works end-to-end â€” the `UIStateChange::ActivePerspective` event is emitted, the frontend subscribes via `listen(\"ui-state-changed\", ...)` in `ui-state-context.tsx`, and `PerspectiveProvider` derives `activePerspective` from `active_perspective_id`. So the event/render pipeline is fine â€” only the keybindingâ†’dispatchâ†’arg-injection path is missing.
