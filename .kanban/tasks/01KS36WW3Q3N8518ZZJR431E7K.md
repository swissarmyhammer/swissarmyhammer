## What

Rewrite the frontend's `useDispatchCommand` hook so it calls the Command MCP server's `execute command` verb instead of the existing Rust-side `swissarmyhammer-commands` registry. The hook's signature stays the same (`useDispatchCommand(commandId: string)` returning a dispatch function) — only the backend changes.

Files (in `apps/kanban-app/`):
- `apps/kanban-app/ui/src/hooks/useDispatchCommand.ts` — rewrite. Today it likely calls a Rust dispatcher via `mcp_call("commands", ...)` or a dedicated Tauri command; new impl calls `tools/call("command", { op: "execute command", id, ctx })` through the existing MCP transport.
- `apps/kanban-app/ui/src/hooks/useCommandList.ts` (new or existing) — hook that calls `list command` with scope filters and subscribes to `notifications/commands/changed` so the palette/menu re-renders on registry updates.

Memory `commands-in-rust`: command logic lives in Rust/plugin code, not in React. Memory `useDispatchCommand-signature`: hook takes a command name string. Memory `no-module-level-dispatch`: never preserve module-level dispatch — components consume the hook directly.

Context resolution: today's dispatcher resolves `target`, `scope_chain`, and `args` server-side using the React context. New world: the hook captures the current scope chain + target + args at call time and passes them in the `ctx` parameter of `execute command`. The Command service forwards them to the registered callback unchanged; the callback (in the plugin) reads what it needs.

## Acceptance Criteria
- [ ] `useDispatchCommand("task.move")` (and every other built-in id) dispatches through the Command service
- [ ] The hook's external API is unchanged — no React component changes required
- [ ] `useCommandList({ scope })` returns the current set of commands matching scope and re-renders on `notifications/commands/changed`
- [ ] No frontend code path bypasses the Command service for user-invocable commands
- [ ] Stale `notifications/commands/changed` debouncing on the frontend matches the server's (~100ms)
