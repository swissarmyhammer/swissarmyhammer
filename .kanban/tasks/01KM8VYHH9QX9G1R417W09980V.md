## What

`save_window_geometry` is registered as a Tauri command but the frontend never calls it. Window geometry is saved directly from the `on_window_event` handler in `main.rs` via `state.ui_state.save_window_geometry()`. The Tauri command is dead code.

### Changes
- Remove `save_window_geometry` function from `commands.rs`
- Remove it from `invoke_handler` in `main.rs`
- Update the flowerbox comment if it references this command
