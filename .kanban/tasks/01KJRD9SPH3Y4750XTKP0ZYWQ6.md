**File:** `swissarmyhammer-kanban/src/context.rs` (throughout)\n\n**What:** The context.rs file contains 35 methods marked `#[deprecated]` with notes pointing to `entity_context()` replacements. All command modules (actor, tag, column, swimlane, board, task) have been migrated to use `entity_context()` directly. The only remaining production caller of deprecated methods is `KanbanLookup` in `defaults.rs` (see separate card).\n\nThese deprecated methods represent ~400 lines of legacy I/O code that is no longer needed by any active command. They include:\n- `read_board` / `write_board`\n- `first_column` / `terminal_column` / `find_column` / `find_swimlane`\n- `read_task` / `write_task` / `read_all_tasks` / `list_task_ids` / `delete_task_file`\n- `read_actor` / `write_actor` / `read_all_actors` / `list_actor_ids` / `delete_actor_file` / `actor_exists`\n- `read_tag` / `write_tag` / `read_all_tags` / `list_tag_ids` / `delete_tag_file` / `tag_exists` / `find_tag_by_name` / `tag_name_exists`\n- `read_column` / `write_column` / `read_all_columns` / `list_column_ids` / `delete_column_file` / `column_exists`\n- `read_swimlane` / `write_swimlane` / `read_all_swimlanes` / `list_swimlane_ids` / `delete_swimlane_file` / `swimlane_exists`\n\nAlso: the `TaskMeta` struct and `parse_task_markdown()` helper exist solely to support the deprecated `read_task` / `write_task`.\n\n**Why it matters:** Per the user's preference, migration code should be deleted, not kept around. These deprecated methods add maintenance burden, increase compile time, and obscure the actual API surface.\n\n**Suggestion:**\n- [ ] First fix the `KanbanLookup` task branch (see other card) so no production code calls deprecated methods\n- [ ] Remove all deprecated methods from `KanbanContext`\n- [ ] Remove `TaskMeta` struct and `parse_task_markdown()` helper\n- [ ] Remove unused imports (`Comment`, `Attachment`, etc. from the use block at the top)\n- [ ] Update tests that reference deprecated methods to use `entity_context()`\n- [ ] Run full test suite to verify nothing breaks\n\n#warning #warning
