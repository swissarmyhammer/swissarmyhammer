## What
Create `code-context-cli/.mcp.json` — the local MCP server config that ships with the crate, mirroring `shelltool-cli/.mcp.json`.

Content:
```json
{
  "mcpServers": {
    "code-context": {
      "command": "code-context",
      "args": ["serve"]
    }
  }
}
```

Note: the `.mcp.json` in shelltool-cli is malformed (two top-level objects). The code-context version must be valid JSON.

This file lets users drop into the `code-context-cli/` directory and have the MCP server auto-detected by their agent.
