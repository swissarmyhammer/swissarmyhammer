## What

Each paste handler card writes its own `KanbanContext` setup, entity fixtures, and `CommandContext` assembly. After all 7 land, that's ~300 LOC of duplicated test-support code. Extract the common helpers into `swissarmyhammer-kanban/src/commands/paste_handlers/test_support.rs` and refactor the handler tests to use them.

### Helpers to extract

- `setup_board_with_columns(&[&str])` → `(TempDir, KanbanContext, Vec<ColumnId>)`
- `create_task(&ctx, title, column)` → `TaskId`
- `create_tag(&ctx, name)` → `TagId`
- `create_project(&ctx, name)` → `ProjectId`
- `create_actor(&ctx, id, name)` → `ActorId`
- `clipboard_with(ui, entity_type, entity_id, fields)` — populate `UIState::clipboard_payload` for tests
- `dispatch_paste(&ctx, &ui, target_moniker)` → `Result<Value>`
- Assertion helpers: `assert_task_in_column`, `assert_task_has_tag`, `assert_task_has_assignee`, etc.

### Files to touch

- CREATE `swissarmyhammer-kanban/src/commands/paste_handlers/test_support.rs` with the helpers.
- MODIFY each of the 7 paste handler source files — refactor their `#[cfg(test)]` modules to use the shared helpers. Delete local duplicates.
