**Severity**: Low (pattern consistency)\n**Layer**: Design / Pattern following\n**Files**: `swissarmyhammer-perspectives/src/context.rs:77`, `swissarmyhammer-entity/src/context.rs:218`\n\nIn `PerspectiveContext::write()`, the store handle delegation uses:\n```rust\nsh.write(perspective).await.map_err(PerspectiveError::Store)?;\n```\n\nBut `EntityContext::write()` uses a bare `?`:\n```rust\nlet entry_id = sh.write(&entity).await?;\n```\n\nThis works for EntityContext because `EntityError` has `#[from] swissarmyhammer_store::StoreError`, which gives the blanket `From` conversion.
