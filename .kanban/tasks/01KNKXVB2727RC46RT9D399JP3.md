### New examples
Replace with DSL filter examples:
```json
{\"op\": \"next task\", \"filter\": \"#review-finding\"}
{\"op\": \"list tasks\", \"column\": \"todo\"}
{\"op\": \"list tasks\", \"filter\": \"#bug\"}
{\"op\": \"list tasks\", \"filter\": \"#READY\"}
{\"op\": \"list tasks\", \"filter\": \"#bug && @alice\"}
{\"op\": \"list tasks\", \"filter\": \"#bug || #feature\"}
{\"op\": \"list tasks\", \"filter\": \"!#done && #READY\"}
```

### Filter DSL reference to add
Brief reference section explaining the sigils and operators:
- `#tag` — filter by tag (includes virtual tags: READY, BLOCKED, BLOCKING)
- `@user` — filter by assignee
- `^card-id` — filter by card reference/dependency
- `&&` / `and` — both must match
- `||` / `or` — either must match
- `!` / `not` — negate
- `()` — grouping
- Adjacent atoms without operator = implicit AND
