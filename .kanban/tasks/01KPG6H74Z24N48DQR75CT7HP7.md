## What

Implement `ColumnIntoBoardHandler` — duplicates a column into a target board. Handler at `swissarmyhammer-kanban/src/commands/paste_handlers/column_into_board.rs`, matches `("column", "board")`.

### Action

1. Parse board id from target moniker.
2. Clone `clipboard.fields` (column's `name`, `order`, any styling).
3. Drop the source `order` — new column is appended at the end (highest order + 1) unless a later product decision changes this.
4. Invoke `AddEntity::new("column").with_overrides(fields)`.
5. Do NOT copy the column's tasks — this handler duplicates the column structure only. Task duplication is a separate concern.
6. If `clipboard.is_cut`, delete the source column via `DeleteColumn::new(&clipboard.entity_id)`.
