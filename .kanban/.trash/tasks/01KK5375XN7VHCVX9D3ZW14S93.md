## What
Create the `swissarmyhammer-lsp` crate with types, the built-in server registry, and public API surface. No daemon logic yet — just the data model and registry that maps `ProjectType` → `LspServerSpec`.

Files: `swissarmyhammer-lsp/Cargo.toml`, `swissarmyhammer-lsp/src/lib.rs`, `swissarmyhammer-lsp/src/registry.rs`, `swissarmyhammer-lsp/src/types.rs`
