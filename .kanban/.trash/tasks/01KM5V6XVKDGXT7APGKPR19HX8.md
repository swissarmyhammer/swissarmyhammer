## What\n\nCurrently `App.tsx` manages `panelStack` as local React state via `inspectEntity`, `closeTopPanel`, `dismissTopPanel`, and `closeAll` callbacks that directly call `setPanelStack`. These bypass the command architecture entirely.\n\nRefactor so that:\n1. `inspectEntity` calls `dispatch_command` with `ui.inspect` instead of manipulating local state\n2. `dismissTopPanel` calls `dispatch_command` with `ui.inspector.close`\n3. `closeAll` calls `dispatch_command` with `ui.inspector.close_all`\n4. `panelStack` is driven by listening to `ui-state-changed` events (from Card 1)\n\n### Files\n- `kanban-app/ui/src/App.tsx` — replace direct state manipulation with dispatch_command calls + event listener\n\n### Approach\n- Replace `inspectEntity` callback: instead of `setPanelStack(...)`, call `invoke('dispatch_command', { cmd: 'ui.inspect', target: moniker })`\n- Replace `dismissTopPanel`: call `invoke('dispatch_command', { cmd: 'ui.inspector.close' })`\n- Replace `closeAll`: call `invoke('dispatch_command', { cmd: 'ui.inspector.close_all' })`\n- Add `useEffect` listening to `ui-state-changed` event — when payload is `InspectorStack`, update `panelStack` from the event data\n- Remove the `useEffect` that calls `set_inspector_stack` (persistence now happens server-side in Card 1)\n- Keep `get_ui_context` restoration on mount (still needed)\n\n## Acceptance Criteria\n- [ ] Opening an inspector goes through `dispatch_command('ui.inspect', ...)`\n- [ ] Closing inspector goes through `dispatch_command('ui.inspector.close')`\n- [ ] Close-all goes through `dispatch_command('ui.inspector.close_all')`\n- [ ] `panelStack` updates reactively from `ui-state-changed` events\n- [ ] Direct `set_inspector_stack` call removed\n- [ ] Inspector stack still restores on mount/hot-reload\n\n## Tests\n- [ ] `kanban-app/ui/src/App.test.tsx` or equivalent — verify inspect calls dispatch_command\n- [ ] `npx vitest run` passes\n\n## Subtasks\n- [ ] Replace `inspectEntity` with dispatch_command call\n- [ ] Replace `dismissTopPanel` / `closeAll` with dispatch_command calls\n- [ ] Add `ui-state-changed` event listener to drive panelStack\n- [ ] Remove `set_inspector_stack` persistence effect\n- [ ] Verify mount restoration still works\n\n## depends_on\n- Card 1: Emit ui-state-changed Tauri event from dispatch_command"
