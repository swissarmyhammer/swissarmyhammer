Create `ui/src/components/fields/field-editor.tsx` — a single dispatcher that routes on `field.editor` (from YAML config) with a `mode` parameter (`compact` for grid inline edit, `full` for inspector).

Dispatches:
- `markdown` → CM6 editor (FieldPlaceholderEditor in compact/single-line, EditableMarkdown in full/multi-line)
- `select` → SelectEditor (dropdown, same in both modes)
- `multi-select` → MultiSelectEditor (checkbox list or tag picker)
- `number` → NumberEditor (CM6-wrapped or styled input, NOT plain `<input type=number>`)
- `date` → DateEditor (date picker component)
- `color-palette` → ColorPaletteEditor (HexColorPicker in full, compact palette in compact)
- `none` → no editor rendered (read-only field)

Interface:
```tsx
interface FieldEditorProps {
  field: FieldDef;
  value: unknown;
  entity: Entity;
  mode: "compact" | "full";
  onCommit: (value: unknown) => void;
  onCancel: () => void;
}
export function FieldEditor({ field, value, entity, mode, onCommit, onCancel }: FieldEditorProps)
```

CRITICAL: All text editing MUST use CM6 (CodeMirror 6) — never plain `<input type="text">`. Number/date/color can use styled native inputs since CM6 doesn't help there.
