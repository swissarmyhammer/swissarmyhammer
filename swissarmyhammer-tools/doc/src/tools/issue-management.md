# Issue Management

Git-friendly work item tracking using markdown files with YAML frontmatter.

## Available Tools

All issue tools are documented in the [Tools Overview](overview.md).

Issues are stored in `.swissarmyhammer/issues/` as markdown files, making them:
- Git-friendly and version-controlled
- Human-readable
- Easy to review and share
- Integrated with your development workflow
